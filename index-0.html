사람 vs 마이크로컨트롤러

MCU는 사람의 신체 구조와 매우 비슷하게 작동합니다.

사람 (Human)

  • 입력 (Input) 눈, 귀, 코 (감각기관)
  • 처리 (Process) 뇌 (Brain)
  • 출력 (Output) 손, 발, 목소리 (반응)

마이크로컨트롤러 (MCU)

  • 입력 (Input) 스위치, 센서
  • 처리 (Process) CPU (중앙처리장치)
  • 출력 (Output) LED, 모터, 부저

MCU의 3대 요소

마이크로컨트롤러를 구성하는 핵심 요소들을 알아봅시다.

입력 (Input)

외부의 상태를 읽어들이는 과정입니다. 버튼이 눌렸는지, 온도가 몇 도인지 확인합니다.

센서, 스위치

처리 (Processing)

입력된 정보를 바탕으로 판단하고 계산합니다. "만약(If) 버튼이 눌리면"

CPU, 메모리

출력 (Output)

판단 결과에 따라 외부 장치를 제어합니다. LED를 켜거나 모터를 돌립니다.

액추에이터

디지털 vs 아날로그

MCU가 세상을 이해하는 두 가지 언어입니다.

디지털 신호 (Digital)

ON (1, High, 5V) OFF (0, Low, 0V)

딱 두 가지 상태만 존재합니다. 전등 스위치와 같습니다.

아날로그 신호 (Analog)

연속적인 값 (0 ~ 1023)

소리의 크기나 빛의 밝기처럼 연속적으로 변하는 값입니다.

가상 실습: LED 제어하기

버튼을 눌러 MCU에 입력 신호를 보내고, 출력이 어떻게 변하는지 확인해보세요.

입력 장치 (Input)

ATmega328P
if (button == HIGH) led = ON; else led = OFF;
